Guía Interactiva: Despliegue de Aplicaciones Angular con Docker

1. Requisitos Previos

Sistema operativo: Windows 10 64-bit (Pro, Enterprise o Education) o Windows 11

Virtualización habilitada en la BIOS

2. Habilitar WSL (Windows Subsystem for Linux)

Actualiza WSL manualmente desde PowerShell (como Administrador)

Ejecuta este comando para actualizar el componente WSL al más reciente:

wsl --update

Reinicia WSL
Después de la actualización, reinicia WSL con este comando:

wsl --shutdown

Luego ejecuta:

wsl --version

Debes ver algo como:

3. Descargar Docker Desktop

Ve a la página oficial:

Descargar Docker Desktop

Haz clic en "Download for Windows".

4. Construye tu proyecto Angular en modo producción

Esto crea la carpeta que vamos a copiar al contenedor Docker.

ng build --configuration production

5. Crea el archivo Dockerfile en la raíz del proyecto

Crear el Dockerfile en la raíz del proyecto
Este archivo no tiene extensión y se llama exactamente así: Dockerfile

6. Construye la imagen Docker

Abre la terminal y ubícate en la raíz del proyecto, donde está el Dockerfile

Ejecuta:

docker build -t proyectoambientescomputacionales .

7. Ejecuta el contenedor

Ejecuta este comando para iniciar el contenedor:

docker run -d -p 4200:80 proyectoambientescomputacionales

8. Acceder a tu aplicación

¡Tu aplicación está lista en: http://localhost:4200!
